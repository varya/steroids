Welcome to Steroids

To restart your current app or scan another app, tap and hold on the screen with three fingers (one finger on Android). This opens the Scanner context menu.

This file is located at www/index.html. If you have an existing PhoneGap project you can place it in the www/ folder.

Next steps

Edit the application config in config/application.coffee to change settings for your app or to enable tab bar navigation. See www/config.ios.xml for PhoneGap/Cordova settings.
